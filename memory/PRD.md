# LeazeSure Landing Page PRD

## Original Problem Statement
Design a premium, modern SaaS landing page for LeazeSure, inspired by Canva-style websites, with its own unique identity. The website must feel classy, minimal, trustworthy, and simple. Visitors should instantly understand that LeazeSure helps tenants build a reusable Rental Passport, report rent to Equifax, and helps landlords pre-screen and manage tenants faster.

## User Personas
1. **Tenants** - Renters who want to build credit, create reusable rental profiles, and streamline applications
2. **Landlords** - Property owners/managers who need to pre-screen, manage tenants, and report rent payments

## Core Requirements (Static)
- Premium SaaS landing page aesthetic
- Brand colors: #0BFFC9 (Cyan), #0009B3 (Deep Blue), #FFFFFF, #000000
- Fonts: Poppins (headers), Archivo Narrow (body)
- CTAs as buttons only (links to be added later)
- HTML/CSS illustrative mockups (not images)
